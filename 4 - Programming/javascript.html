JavaScript

Just like CSS, JavaScript can be added to HTML documents in a few ways.

JavaScript within a <script> tag, but not in a function will run on page load.
Code within a function will run when that function is called. (See head element of this document)

What can JavaScript do?
    - Change HTML content.
    - Change HTML attribute values.
    - Change HTML styles (CSS).
    - Can hide and show HTML elements.

Let's look at how to define a button callback with the different methods of defining JavaScript.


In the <head> element.

You can define a button's callback with a script tag and function in the document's head element.

This requires the addition of a script tag with an src attribute in the head element of the document.

The script tag looks like this:

<script>
function headCallback(){
	document.getElementById("headID").innerHTML = "After Change!"
}
</script>

HTML Event

You can define a button's callback with an HTML event like this:
<button onclick="document.getElementById(&#34;attributeID&#34;).innerHTML = &#34After Change!&#34;">Button Text</script>

The result looks like this:

Before Change

Some common HTML events:
    - onchange    - Triggers when an HTML element has been changed.
    - onclick     - Triggers when the user clicks the element.
    - onmouseover - Triggers when the user moves the mouse over the element.
    - onmouseout  - Triggers when the user moves the mouse away from the HTML element.
    - onkeydown   - Triggers when the user pushes a keyboard key.
    - onload      - Triggers when the browser has finished loading the page.

Externally

You can define a button's callback with JavaScript in an external file.

This requires the addition of a script tag with an src attribute in the head element of the document.

The script tag looks like this:
<script src="fileName.js"></script>

The result looks like this:

Before Change

External references can take a few different forms:
    - Full URL                    - <script src="https://www.w3schools.com/js/myScript1.js"></script>
    - Specified folder            - <script src="/js/myScript1.js"></script>
    - Same folder as current page - <script src="myScript1.js"></script>
