You can format text with tags.


The "b" tag defines bold text.

The "i" tag defines italic text.

There are many others like:

The <strong> tag for important text,
The <em> tag for emphasized text,
The <mark> tag for highlighted text,
<small> for smaller letters,
<del> for strikethrough text,
<ins> for underlines,
and <sup> for superscript, and <sub> for subscript.
