CSS

CSS stands for Cascading Style Sheet. CSS is used in html and web design to help define how things might look.

CSS can be added to HTML elements in 3 ways.

 - Inline - By using style attributes in the HTML elements.
 - Internal - By using a <style> element in the <head> section.
 - External - By writing the CSS in an external file.

If you are only working with a couple style modifications on only a few elements, inline will work fine. However, If you are applying
several modifications to an element, you will want to get into internal or external CSS.


Inline

Inline CSS can be added by using attributes.

If you wanted to center align a paragraph element, you would use the align attribute with a value of "center".

The center aligned paragraph would look like this: <p align="center">Paragraph text here.<p>

Internal

CSS can be defined internaly inside of an HTML document.

To assign internal style to an HTML element, use the id attribute.

The center aligned paragraph would look like this: <p id="centerAlign">Paragraph text here.<p>

Defining style for a tag internally will automatically apply to all elements of that type in the document.

External

CSS can be defined externally, outside of an HTML document.

To assign external style to an HTML element, use the class attribute.

The center aligned paragraph would look like this: <p class="centerAlign">Paragraph text here.<p>
